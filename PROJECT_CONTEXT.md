# 项目持续记忆

最后更新：2026-07-23

## 用户与项目目标

- 用户计划建立一家由自己100%持有和管理的美国公司，以中国供应链为基础，面向美国提供产品销售、贸易、市场进入和商业解决方案。
- 2026年第一条业务聚焦工业激光清洗设备，目标客户仅限滚塑、塑料加工、橡胶加工及模具相关企业。
- 核心原则：先验证，再投入；先跑通一个业务，再扩展。
- 网站必须是一套日常可打开、执行、勾选、复盘和做阶段决定的创业运营系统，而不是展示型商业计划。
- 法律、税务、移民、产品合规、进口、保险和合同内容只提供框架，必须由专业人士确认。

## 已确认的产品决策

- 技术栈：React + Vite，纯静态网站，无后端、数据库、登录或大型UI框架。
- 数据范围：时间线、SOP、CRM、费用、周复盘、决策门、学习资源、能力评估和模板草稿均保存在浏览器 localStorage。
- 数据可移植性：支持完整 JSON 导入和导出；不同访问者默认看不到彼此的本地数据。
- 部署方式：独立公开 GitHub 仓库和 GitHub Pages，不覆盖旧站。
- 视觉方向：暖白/米白、深灰棕文字、少量金色、安静理性、卡片化、桌面侧栏和移动端菜单。

## 当前已实现系统

- 首页自动显示当前阶段、最近决策门、阶段进度、CRM到期动作和动态学习指标。
- 18个2026执行阶段、5个决策门及分阶段资金授权。
- CRM支持完整客户字段、漏斗、七项资格评分和本地持久化。
- 费用台账区分个人/公司、预算阶段、币种、人民币等值和票据状态。
- 周度工作台支持一个成果、三项行动、障碍、客户/财务数据和假设复盘。
- 18个SOP，包括“SOP 18｜个人学习与能力转化”。
- 工作模板为站内可填写工具，支持自动保存、完成比例、复制结果和清空草稿。
- 独立“个人学习与企业家能力发展”章节，包括：
  - A/B/C学习优先级；
  - 2026—2027学习时间线；
  - 可编辑学习资源库；
  - 同时最多一个主要课程和一个辅助主题；
  - 学习转化闭环；
  - 10项企业家行为；
  - 12项带行为证据的月度能力评估。
- 学习不能挤占客户行动；普通周学习总时间不超过3—4小时。

## 内容一致性规则

- 目标企业：30—50家。
- 有效访谈：10—15次。
- 合格机会：3—5家；决策门最低要求为至少3家。
- 愿意演示或试用：执行目标2—3家；决策门最低要求至少2家。
- 现场演示：基准2—3次。
- 生产试用：至少1次，基准1—2次。
- 资金三阶段：10—20、25—45、30—60万元人民币；第一年授权上限65—125万元。
- Rotoplas现场日期：2026-11-17至11-19；72小时复盘截止：2026-11-22。
- 决策门结论只能是继续、调整或暂停，不能使用“再看看”。
